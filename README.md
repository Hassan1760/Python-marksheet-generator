# Student Marksheet Calculator

*Author:* Hassan Ali  
*Language:* Python  
*Project:* Marksheet generator  
*Date:* November 2025  

## Description

A simple Python program that calculates a student's *total marks, percentage, and grade* based on entered subject marks and displays a formatted *Marksheet*.

## Features

- Input *student information*: Name, Roll Number, Class  
- Input *marks for multiple subjects*  
- Calculate *total, percentage, and grade*  
- Display formatted *Marksheet*
